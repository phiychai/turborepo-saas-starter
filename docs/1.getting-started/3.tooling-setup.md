This project uses a shared configuration approach for ESLint, Prettier, and TypeScript across the monorepo.

## Quick Overview

- **ESLint**: Shared config in `packages/eslint-config/` with app-specific overrides
- **Prettier**: Consistent formatting across all packages
- **TypeScript**: Project references for incremental builds

## Common Commands

```bash
# Lint all packages
pnpm lint

# Auto-fix linting issues
pnpm lint:fix

# Format all files
pnpm format

# Type check all packages
pnpm type-check
```

## Structure

```
packages/eslint-config/     # Shared ESLint/Prettier configs
packages/shared-config/      # Shared TypeScript base config
```

All apps extend these shared configurations with minimal overrides.
